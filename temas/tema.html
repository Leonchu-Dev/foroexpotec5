Consulta sobre Base de Datos para Proyecto Final de 7mo (Informática)

JP
Juan Pérez Publicado hace 2 horas

¡Hola a todos! Estoy empezando a planificar mi proyecto final, que es un sistema de gestión para la biblioteca de la escuela. Tengo dudas sobre qué tipo de base de datos usar.

¿Alguien tiene experiencia con MongoDB para este tipo de proyectos, o me recomiendan seguir con algo relacional como MySQL por ser más tradicional y robusto para manejar relaciones entre libros, autores y préstamos? ¡Gracias!

Respuestas

MG
María Gómez Publicado hace 1 hora

¡Buena pregunta, Juan! Para un sistema de gestión de biblioteca, una base de datos relacional como MySQL o PostgreSQL es una apuesta segura. Es muy robusta para manejar las transacciones de préstamos y devoluciones, y la integridad de los datos es clave. Además, hay muchísima documentación y es lo que más se ve en el mundo laboral.

CR
Carlos Rodriguez Publicado hace 30 minutos

Yo usé MongoDB el año pasado para un proyecto de un blog y la flexibilidad fue increíble. Si prevés que los datos de los libros (como agregar campos nuevos tipo "ebook disponible" o "etiquetas") pueden cambiar mucho, no tener un esquema rígido te salva tiempo. La curva de aprendizaje no es tan alta como parece. ¡Anímate a probarlo si quieres aprender algo nuevo y diferente!
