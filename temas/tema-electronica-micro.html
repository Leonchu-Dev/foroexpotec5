¿Cómo elegir el microcontrolador adecuado (PIC vs Arduino vs ESP32)?

MS
Martín Sosa Publicado hace 5 días

Para un proyecto de domótica simple (controlar luces por WiFi), estoy dudando entre usar un PIC, un Arduino Uno con un módulo WiFi, o directamente un ESP32. ¿Qué me recomiendan en términos de facilidad, costo y potencia?

Respuestas

CR
Carlos Rodriguez Publicado hace 4 días

¡Hola Martín! Para lo que quieres hacer, el ESP32 es la mejor opción por lejos. Ya tiene WiFi y Bluetooth integrados, es muy potente y se puede programar con el mismo IDE de Arduino, así que la curva de aprendizaje es casi nula si ya conoces Arduino. Un PIC sería complicarse demasiado para este proyecto y el Arduino + módulo te saldría más caro y sería más grande.

JG
Prof. Juarez Publicado hace 4 días

Coincido con Carlos. El ESP32 es la plataforma ideal para proyectos de IoT (Internet de las Cosas) hoy en día. Aprender a usarlo te dará una herramienta muy potente para futuros proyectos. Si bien aprender a programar un PIC en ensamblador o C es un gran ejercicio académico, para una solución rápida y funcional, el ESP32 es el camino.
