Optimización de un servidor web Apache para un proyecto de 6to

ST
Sofía Torres Publicado hace 6 días

Hola, para el proyecto de Aplicaciones Web tenemos que montar un sitio en un servidor local con Apache. El sitio tiene bastantes imágenes y notamos que a veces carga un poco lento.

¿Hay alguna configuración básica en Apache que podamos tocar para mejorar el rendimiento? ¿Activar la compresión gzip o algo así?

Respuestas

JP
Juan Pérez Publicado hace 5 días

¡Totalmente! Activar `mod_deflate` para la compresión GZIP es el primer paso y hace una diferencia enorme. También puedes mirar `mod_expires` para decirle al navegador que guarde en caché los archivos estáticos (CSS, JS, imágenes) por un tiempo. Con esas dos cosas, la velocidad de carga va a mejorar muchísimo.
