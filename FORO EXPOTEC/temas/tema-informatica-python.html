Ayuda con error en bucle `for` en Python

LF
Lucía Fernandez Publicado hace 1 día

Hola, estoy tratando de iterar sobre una lista y modificarla al mismo tiempo, pero me da resultados raros. A veces se salta elementos. Este es mi código:

numeros = [1, 2, 3, 4, 5, 6]
for n in numeros:
    if n % 2 == 0:
        numeros.remove(n)
print(numeros) # Esperaba [1, 3, 5] pero imprime [1, 3, 5, 6]

¿Por qué pasa esto?

Respuestas

ST
Sofía Torres Publicado hace 20 horas

¡Hola Lucía! Es un error muy común. Nunca debes modificar una lista sobre la que estás iterando directamente. Cuando eliminas un elemento, la lista se "achica" y el bucle se salta el siguiente elemento porque los índices se corren.

La forma correcta de hacerlo es iterar sobre una copia de la lista, o crear una lista nueva con los elementos que quieres conservar. Por ejemplo:

numeros = [1, 2, 3, 4, 5, 6]
numeros_impares = []
for n in numeros:
    if n % 2 != 0:
        numeros_impares.append(n)
print(numeros_impares) # Esto sí imprime [1, 3, 5]
